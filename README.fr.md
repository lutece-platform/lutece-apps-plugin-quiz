# Plugin quiz

## Introduction

Ce plugin permet de créer facilement des Quiz ou questionnaire à choix multiples.

La présentation et du texte éditorial peuvent être gérés à chaque question.

Les bonnes réponses peuvent être détaillées au moment du calcul du score final.

Les données des réponses peuvent être exportées vers différentes cibles (fichier CSV, base de données, ...)
